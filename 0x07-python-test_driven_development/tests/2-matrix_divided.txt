# matrix_divided.txt

==================================
How to Use matrix_divided.py
==================================

This module defines a function `matrix_divided(matrix, div)` to divide all elements of a matrix.

Usage
=====

The `matrix_divided` function divides all elements of the given matrix by the specified divisor `div` and returns a new matrix with the result.

```python
>>> matrix_divided = __import__('matrix_divided').matrix_divided
>>> matrix = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
>>> div = 2
>>> result = matrix_divided(matrix, div)
>>> result
[[0.5, 1.0, 1.5], [2.0, 2.5, 3.0], [3.5, 4.0, 4.5]]
>>> matrix = [[1, 2, 3], [4, 5, 6]]
>>> div = 3
>>> result = matrix_divided(matrix, div)
>>> result
[[0.33, 0.67, 1.0], [1.33, 1.67, 2.0]]

===================
Matrix Requirements
===================
The matrix argument must be a list of lists of integers or floats, 
otherwise, raise a TypeError exception with the message "matrix must be a matrix (list of lists) of integers/floats".

>>> matrix = [[1, 2, 3], [4, 5, 6], [7, 'a', 9]]
>>> div = 2
>>> matrix_divided(matrix, div)
Traceback (most recent call last):
TypeError: matrix must be a matrix (list of lists) of integers/floats

Each row of the matrix must be of the same size, 
otherwise raise a TypeError exception with the message "Each row of the matrix must have the same size".

>>> matrix = [[1, 2, 3], [4, 5], [7, 8, 9]]
>>> div = 2
>>> matrix_divided(matrix, div)
Traceback (most recent call last):
TypeError: Each row of the matrix must have the same size

====================
Divisor Requirements
====================
The div argument must be a number (integer or float), otherwise raise a TypeError exception with the message "div must be a number".
>>> matrix = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
>>> div = 'hello'
>>> matrix_divided(matrix, div)
Traceback (most recent call last):
TypeError: div must be a number

The div argument can't be equal to 0, otherwise raise a ZeroDivisionError exception with the message "division by zero".
>>> matrix = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
>>> div = 0
>>> matrix_divided(matrix, div)
Traceback (most recent call last):
ZeroDivisionError: division by zero
